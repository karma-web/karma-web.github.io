管道操作符

该功能只有 Firefox 58 及以上版本通过传入启动参数来支持，不过 Babel 已经有一个针对它的 插件提议

HTMLElement.dataset

它的作用基本就是在 JS 中查询的时候允许你使用驼峰命名法（camelCase）来查询「减号-命名」（dash-case）的属性。所以属性名 data-birth-planet 在 JS 中就变成了 birthPlanet。
